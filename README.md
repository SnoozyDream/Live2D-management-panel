# Live2D Asset Management Panel (Portfolio)

Live2Dモデルアセットとユーザー権限を管理するためのWebベース管理画面のプロトタイプです。

## 🚀 プロジェクト概要
所属ライバーのLive2Dモデル（.moc3）やテクスチャ、モーションデータを一括管理し、ブラウザ上でプレビュー・更新を行うためのシステムを想定しています。
現在はフロントエンドの基盤およびアセットのプレビュー機能を実装中です。

## ✨ 実装済み機能
- **ダッシュボードUI**: 管理ツールとして使いやすいサイドメニュー形式のレイアウト。
- **アセットプレビュー**: `FileReader API` を活用し、選択した画像をサーバー保存前にブラウザ上で即座に確認可能。
- **レスポンシブ対応**: PCおよびモバイルブラウザからの閲覧に対応。

## 🛠 使用技術
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Deployment**: GitHub Pages
